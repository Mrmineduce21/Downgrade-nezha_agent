# 怎么使用


## Debian / Ubuntu 安装下载工具

```
apt update -y  && apt install -y curl
```

## CentOS 安装下载工具

```
yum update && yum install curl -y
```

## Alpine 安装下载工具

```
apk update

apk add curl
```
